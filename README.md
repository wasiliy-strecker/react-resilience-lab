# React Resilience Lab

Production-minded React patterns for cancellation, race-free data fetching,
optimistic updates, conflict recovery, offline replay, and accessible failure
states.

The lab will use an incident operations console and an intentionally unreliable
Node.js API to make failure behavior reproducible. Each pattern will be backed
by executable tests and explicit guarantees rather than happy-path screenshots.

## Status

Initial project setup. The first implementation milestone will establish the
typed workspace, shared HTTP contracts, CI, and the baseline console.
