# Changelog

All notable changes to this project are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and releases use
[Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-22

### Added

- React incident console with explicit loading, refresh, stale, empty, and
  recovery states
- deterministic Express fault API with runtime-validated problem details
- conditional and idempotent incident command protocol
- generic persistent command outbox with memory and IndexedDB adapters
- write-ahead optimistic projections, ordered replay, and conflict isolation
- explicit keep-or-rebase conflict recovery and render error boundary
- integrated Playwright outage, recovery, and accessibility proofs
- multi-runtime CI and tagged GitHub release automation
